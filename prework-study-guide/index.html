HTML

  • The head element contains information about the webpage.
  • The head element contains information that is not visible to the user.
  • The body element represents the visible content showin to the user.
  • The footer element is at the bottom of the webpage and usually houses information regarding authors and site maps.

CSS

  • There are three ways to style a webpage using CSS: Inline, Internal, External
  • Inline is styling an element directly into the HTML file
  • Inline applies directly to an HTML tag, styling a particular page element only
  • Inline is useful for quick and permanent changes, but not as flexible as Internal or External
  • Internal contains CSS rules within the HEAD of the HTML section of the HTML file
  • External is a separate file which attaches to the main HTML file
  • External makes design changes easier since all CSS rules are in a separate file
  • A margin indicates how much space we want around the outside of an elemnt
  • A padding indicates how much space we want around the content inside an element
  • A box-shadow is described by X and Y offsets, X axis to the right of the box, Y axis just below the box
  • Colors can be assigned using semantics such as "gray" or "red" and can also be defined by its hexadecimal code or hex #
